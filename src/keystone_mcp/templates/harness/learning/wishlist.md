# Learning wishlist

Team-curated catalog of known gaps in the harness. Items here become
real `learning/inbox/` candidates only when a concrete situation
triggers them — the wishlist itself never promotes to guides directly.

## How to use

Add a bullet describing the gap. Reference the trigger condition that
would convert it from wishlist → inbox.
